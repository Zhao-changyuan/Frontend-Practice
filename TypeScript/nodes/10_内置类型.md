### keyof用法

keyof 用于获取某种类型的所有键，其返回类型是联合类型。

```ts
interface B {
	id: number;
    name: string;
    age: number;
}
type B1 = keyof B;
// type B1 = 'id' | 'name' | 'age'
```

### `Partial<T>`

可以快速吧某个接口类型中定义的属性变成可选的（Optional）

### `Pick<T, K>`

从一个符合类型中，取出几个想要的类型的组合。

### `Record<K, V>`

Record 后面的泛型就是对象键和值的类型；
