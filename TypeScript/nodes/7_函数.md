## 函数类型

设置函数结构的类型声明：

语法：

(形参：类型， 形参：类型 ...) => 返回值

```typescript
let d: (a:number, b: number) => number // 函数类型声明
```

在参数名称后添加 `?` 表示可选参数。



为了使用接口表示函数类型，我们需要个接口定义一个调用签名。

它就像是一个只有参数列表和返回值类型的函数定义，参数列表里的每个参数都需要名字和类型：

```typescript
interface ISearchFunc {
    (source: string, subString: string): boolean;
}
```
