#### 8.11 受控组件与非受控组件

“非受控组件”在React设计中属于另类，是通过与State（状态）无关的Refs概念实现的。对于“非受控组件”，是运行组件直接操作实际DOM的。

非受控组件主要适用于以下几个场景：

- 管理焦点，文本选择或媒体播放
- 触发强制动画
- 继承第三方DOM库
